 # 使用方法
 1、准备数据  
 数据格式为：classifier_multi_label_textcnn/data/test_onehot.csv  
 2、参数设置  
 参考脚本 hyperparameters.py，直接修改里面的数值即可。  
 3、训练  
 python train.py  
 4、预测  
 python predict.py
